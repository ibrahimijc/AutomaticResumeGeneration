Automatic Resume Generation using OpenAI

Project Summary

The purpose of this project is to develop an automated resume generator using artficial intelligence, aimed at simplifying the resume customization process for job seekers tailored for each job description. By leveraging the capabilities of OpenAI's GPT-4 model, the application enables users to input their basic details and generate customized resumes tailored to specific job descriptions.

Novelty of the Project

Innovative Use of AI Employing OpenAI's GPT-4 model specifically for the purpose of customizing resumes based on job descriptions. There are no solutions in the market that tailors resume based on job description. By using this application candidate can successfully generate unique resume tailored on each job description and hence applying to different organiza@on with confidence.

Instructions for Usage

  1. Step 1: In the backend code file GeneratedResumeService.cs, replace the OpenAI key with your own key.
  2. Step 2: Run the backend code using any of the .net IDE Visual Studio and Rider.
  3. Step 3: In the frontend code, change the url for api (resume.depchip.com/) to your own url of the backend application.
  4. Step 4: Run the frontend code with yarn run dev.
